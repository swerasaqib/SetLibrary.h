Project Title: Set Theory Operations Library

Description:
This project implements a custom C++ library based on set theory.
It supports operations such as union, intersection, and difference.
Function overloading is used for adding elements in multiple ways,
and operator overloading is used to perform mathematical set operations.

Sample Output:
Set A: { 1 2 3 10 20 30 3 4 5 }
Set B: { 3 4 5 }
Union (A + B): { ... }
Difference (A - B): { ... }
Intersection (A * B): { ... }
Sets are NOT equal
